# 第一次写个人博客

了解到写个人博客主要使用的是markdown语言，所以今天就在这里写一下markdown语言的用法吧。先写点用法，再插入点历史。


# 一、标题
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```
效果如下：
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

# 二、字体

（1）加粗

要加粗的文字左右用两个**包起来

例:**加粗**

（2）斜体

要倾斜的字体左右分别用一个*包起来

例:*倾斜*

（3）斜体加粗

要加粗的文字左右分别用三个*包起来

例如:***斜体加粗***

（4）删除线

要加删除线的文字左右分别用两个~~包起来

例：~~删除线~~

# 三、引用
在引用的文字前加>即可，引用也可以嵌套，如加两个>>三个>>>

>这是引用的内容

>>这是引用的内容

>>>这是引用的内容

# 四、分割线

三个或者三个以上的-或者*都可以

---
----
***
****

# 五、图片

语法:
```
![图片alt](图片地址)
```

![Wlop](../img/1.png)

# 六、超链接
语法:

```
[超链接名](超链接地址)
```
示例:

[百度](http://baidu.com)

# 七、列表

无序列表

语法:

```
+ - *任何一种都可以
```
示例:


+ 111
- 111
* 111

有序列表

语法:

```
数字加点
```
1. 试验1
2. 试验2
3. 试验3

列表嵌套

上一级与下一级敲三个空格即可

+ 111
   + 啦啦啦啦
   + 嘤嘤嘤
   + 嘻嘻嘻

+ 222
   1. 啦啦啦
   2. 哈哈哈
   3. 嘻嘻嘻

1. 哈哈哈哈
   + 哈哈哈
   + 哈哈哈
   + 哈哈哈


1. 啦啦啦
   1. 哈哈哈哈
   2. 哈哈哈
   3. 哈哈哈

# 八、表格

语法:
```
表头|表头|表头
---|:---:|---:
内容|内容|内容
内容|内容|内容
```

姓名|技能|排行
---|:---:|---:
刘备|哭|大哥
关羽|打|二哥
张飞|骂|三弟

# 九、代码

语法:

单行代码:代码之间分别用一个反引号包起来
`Hello,World`

代码块:代码之间分别用三个反引号包起来

```
   Hello World
   Hello World
   Hello World
```
